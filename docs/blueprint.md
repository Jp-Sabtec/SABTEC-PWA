# **App Name**: SabtecPWA

## Core Features:

- Landing Page - Calendar: Initial landing page with a basic calendar view (empty at MVP stage).
- Left Navigation Menu: A slide-out navigation menu accessible from the left, containing links to: Calendar, Timesheet, and Leave.
- Timesheet Page: Basic timesheet page that will have all of the timesheets populated in date order. Currently empty for MVP.
- Leave Request Page: A page dedicated to creating leave requests, featuring four buttons: Leave Type, Date Start, Date End, and Request Leave. On Submission add Leave details into database, to save.

## Style Guidelines:

- Primary color: Dark blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light grey (#F5F5F5) for a clean and modern look.
- Accent color: A brighter blue (#2196F3) to highlight interactive elements and calls to action.
- Body and headline font: 'Roboto' for a clear and readable user experience.
- Clean and intuitive layout that adapts well to mobile screens.
- Use of clear and simple icons from a standard library (e.g., Material Design Icons) for navigation and actions.
- Subtle transitions and animations for smooth navigation between pages.
- Calendar Page: Display a calendar with a similar style of the image given. Navigation on the left side bar with employee name and photo. Main section will be the calendar with the days displayed and any time off that is currently booked by that employee on the time off. Show the current time line in line with a light aqua/blue color (#7DF9FF) . There will also be a calender legend at the bottom.
- Timesheet Page: The timesheet page should have the same format as the provided image. It will have a Date entry, Hours entry, Billable Status Dropdown, Client Dropdown, Project Dropdown and Description
- Leave Request Page: The leave request page should follow a similar format to the image provided, including elements for 'Select Leave Type', 'Start Date' and 'End Date' with corresponding time selections, and a prominent 'Request Leave' button.
